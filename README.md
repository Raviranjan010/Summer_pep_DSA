## 🧭 How To Study (The 30-Minute Daily Plan)

You don't need hours. **30 honest minutes a day** beats one panicked 5-hour session.

1. **(5 min)** Read the "Explain Like I'm 5" box for today's topic.
2. **(10 min)** Type ONE example by hand. No copy-paste — typing builds memory.
3. **(10 min)** Do a **dry run on paper**: take a tiny input like `[3, 1, 2]` and write each variable's value step by step. This one habit turns "magic" into "obvious."
4. **(5 min)** Solve the linked **Easy** practice problem. Even attempting it counts as a win.

> 🎯 **The only rule:** Be **clear**, not fast. Speed shows up on its own once things make sense.
